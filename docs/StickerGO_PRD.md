# StickerGO 產品需求文件 (Final)

## 1. 核心玩法
- 基於 LBG (地理位置) 的貼圖尋寶。
- 階級：🌟 幻神 > 💎 靈貓 > ✨ 精銳 > 🐾 凡貓。

## 2. 經濟與等級
- 經驗值 (EXP)：尋找/產生貼圖獲得。
- 點數 (Points)：升級獎勵，用於透過 Magic Sticker 召喚新貼圖。
- Pro 機制：保底「精銳」以上等級，EXP 1.2x 加成。

## 3. 技術架構
- Frontend: Flutter
- Backend: Firebase (Auth, Firestore, Storage)
- AI Interface: Magic Sticker API (Unified Points)
